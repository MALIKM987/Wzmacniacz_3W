# Wzmacniacz 3W

## Opis projektu
Wzmacniacz 3W to projekt wzmacniacza audio przygotowany w ramach pracy projektowej z zakresu elektroniki. Repozytorium obejmuje materiały związane z projektowaniem układu, symulacją, przygotowaniem PCB oraz dokumentacją techniczną projektu.

Celem projektu było opracowanie kompletnego rozwiązania obejmującego zarówno część schematową i projektową, jak i analizę działania układu oraz przygotowanie materiałów opisowych.

## Zakres projektu
Projekt obejmuje:
- opracowanie koncepcji wzmacniacza audio o mocy 3 W,
- przygotowanie schematu i projektu PCB,
- wykonanie symulacji układu,
- zebranie materiałów technicznych i literaturowych,
- opracowanie sprawozdania i prezentacji projektu.

## Struktura repozytorium
- `pcb/` – pliki związane z projektem płytki drukowanej,
- `sym/` – pliki symulacyjne i analizy działania układu,
- `sprawozdanie/` – dokumentacja opisująca projekt i jego założenia,
- `prezentacja/` – materiały prezentacyjne,
- `materiały/` – dodatkowe pliki robocze i pomocnicze,
- `literatura/` – źródła i materiały referencyjne.

## Wykorzystane obszary i narzędzia
Projekt dotyczy zagadnień związanych z:
- elektroniką analogową,
- projektowaniem układów audio,
- projektowaniem PCB,
- symulacją obwodów elektronicznych,
- dokumentowaniem procesu projektowego.

## Zakres mojej pracy
W projekcie zajmowałem się:
- przygotowaniem i organizacją dokumentacji projektowej,
- pracą nad częścią schematową i projektową układu,
- analizą działania wzmacniacza w środowisku symulacyjnym,
- opracowaniem materiałów podsumowujących projekt.

## Cel repozytorium
Repozytorium pełni rolę uporządkowanego archiwum projektu, zawierającego pliki projektowe, symulacje oraz dokumentację techniczną.

## Możliwe kierunki rozwoju
- rozszerzenie opisu technicznego układu,
- dodanie wyników pomiarów lub testów,
- uzupełnienie README o schemat blokowy i zrzuty ekranu,
- rozbudowa projektu o bardziej szczegółowy opis zastosowanych komponentów.

# 3W Amplifier

## Project Overview
3W Amplifier is an audio amplifier project prepared as part of an electronics design assignment. The repository contains materials related to circuit design, simulation, PCB preparation, and technical documentation.

The goal of the project was to develop a complete solution covering schematic design, PCB work, circuit analysis, and supporting project documentation.

## Project Scope
The project includes:
- development of a 3 W audio amplifier concept,
- schematic and PCB design preparation,
- circuit simulation and analysis,
- collection of technical references and supporting materials,
- preparation of a report and presentation.

## Repository Structure
- `pcb/` – files related to the PCB design,
- `sym/` – simulation files and circuit analysis,
- `sprawozdanie/` – project report and technical documentation,
- `prezentacja/` – presentation materials,
- `materiały/` – additional working and supporting files,
- `literatura/` – references and source materials.

## Areas and Tools Involved
The project is related to:
- analog electronics,
- audio circuit design,
- PCB design,
- electronic circuit simulation,
- engineering project documentation.

## My Contribution
In this project, I was responsible for:
- preparing and organizing the project documentation,
- working on the schematic and design part of the circuit,
- analyzing amplifier behavior in a simulation environment,
- preparing materials summarizing the project.

## Purpose of the Repository
This repository serves as a structured archive of the project, including design files, simulations, and technical documentation.
